### Ubuntu / PopOS

```bash
sudo pacman -S pandoc pandoc-citeproc pandoc-crossref
```

Or you can install standalone binary build of [pandoc@AUR](https://aur.archlinux.org/packages/pandoc-bin/) without Haskell dependencies.

```bash
yay -S pandoc-bin
```

Timeshift

Wen-Wei Tseng included in Linux Packages
 2020-10-22  45 words   One minute 
Contents
    • Install via apt
    • Install via yay

Timeshift, a system backup and restore manager using rsync or btrfs snapshots.

Install via apt

1

sudo apt install timeshift

Install via yay

Install timeshift@AUR.

1

yay -S timeshift cronie

Then enable cronie service for periodic snapshots in rsync mode.

1

systemctl enable --now cronie.service

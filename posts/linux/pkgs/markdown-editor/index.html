Typora

Typora, a full-featured what-you-see-is-what-you-get Markdown editor.

Install via apt

For Ubuntu and Pop OS.

1
2
3

wget -qO - https://typora.io/linux/public-key.asc | sudo apt-key add -
echo 'deb https://typora.io/linux ./' | sudo tee /etc/apt/sources.list.d/typora.list
sudo apt update && sudo apt install typora

Install via yay

typora@AUR

1

yay -S typora

You may want to also install pandoc to export md to docx.
